Random Vibe Code Webapps

A small collection of games, tools, and webapps, all generated by prompting AIs and rerunning them. Zero hand-written code. Mostly hilarious.

The premise: I never touch the keyboard for code. I prompt, watch the AI work, and iterate based on what breaks. The whole point is to probe where current models thrive and where they fall apart.

Comparison versions across different AIs will land here as I run them.
